# A Collective Mind & Other Documents

The Collective Mind is a group of characters who are linked by a double-layered DID system generated by some cosmic scale lovecraftian yet deeply psychological horror.

Because of each body being part of the overlying system, they feel a massive pull to each other through Empathy. As this is an unavoidable pull, they are all part of a multi-species poly queer platonic relationship. Though, they all have the same skin/fur color of orange.

The original people in the bodies where cast away when the double-layered DID system generated and have been stuck for ~500 years in a mental prison with no sense of time.
